# Лабораторные работы №2 и №3 по информатике (2 семестр)

## Структура проекта
Файлы расположены в корневой директории проекта. Логическое разделение:

### Файлы Лабораторной работы №2
- `dynamic_array.h` — АТД Динамический массив
- `linked_list.h` — АТД Связанный список
- `sequence.h` — АТД Последовательность (базовый интерфейс `Sequence<T>`)
- `array_sequence.h` — `ArraySequence`, `Mutable/ImmutableArraySequence` + CRTP-версии
- `list_sequence.h` — `ListSequence`, `Mutable/ImmutableListSequence`
- `bit.h` — Абстракция бита (`Bit`)
- `bit_sequence.h` — `BitSequence` с побитовыми операциями (AND, OR, XOR, NOT)
- `algorithms.h` — Алгоритмы на последовательностях (статистика, префиксы, инверсии, отражение)
- `tests.cpp`, `tests.h` — Модульные тесты ЛР-2

### Файлы Лабораторной работы №3 (Вариант 16)
- `segment.h` — Структура сегмента кусочной функции `[start, end]: y = kx + m`
- `piecewise_function.h` — Абстрактный базовый класс целевого АТД
- `mutable_piecewise_function.h` — Изменяемая реализация
- `immutable_piecewise_function.h` — Неизменяемая реализация
- `piecewise_enumerator.h` — Энумератор для обхода сегментов
- `piecewise_operators.h` — Перегрузка операторов (`+`, `==`, `<<`, `!=`)
- `piecewise_algorithms.h` — Map/Filter/Reduce над сегментами
- `tests_piecewise.cpp` — Модульные тесты ЛР-3 (~105 кейсов)

### Общие / Инфраструктурные файлы
- `exceptions.h` — Кастомные исключения (`index_out_of_range`, `empty_collection`)
- `option.h` — Тип `Option<T>` для try-семантики
- `sequence_crtp.h` — CRTP-база для статического полиморфизма
- `io_utils.h` — Утилиты вывода в консоль (`operator<<` для последовательностей и функций)
- `main.cpp` — Консольный UI с меню выбора лабораторной и автозапуском тестов
